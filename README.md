# ☸️ DharmaBrowser

A high-performance, lightweight web browser built for speed and privacy. **DharmaBrowser** is designed to run smoothly on any machine, especially systems with 4GB RAM or less.

## 🚀 Key Features
*   **Ultralight Engine:** Optimized to keep memory usage low.
*   **Privacy-Focused:** No bloat, no trackers, just fast browsing.
*   **Minimalist UI:** A clean interface that stays out of your way.
*   **Fast Loading:** Built to handle the modern web without the lag.

## 📥 How to Download
You don't need to clone the code or build it yourself. To get the latest version:

1. Go to the **[Releases](https://github.com)** section on the right side of this page.
2. Download the latest `.exe` or installer file.
3. Run the file and start browsing.

## 🛠️ Performance Tips
For the best experience on low-resource hardware:
*   Keep active tabs to a minimum.
*   Close heavy background apps while browsing.
